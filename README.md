## 🛠️ কীভাবে চালাবেন

```bash
# রিপোজিটরি ক্লোন করুন
git clone https://github.com/your-username/some-python-program.git
cd some-python-program

# যেকোনো প্রজেক্ট চালান
python "​১. সহজ প্রজেক্ট (ভিত্তি তৈরির জন্য)/১. Hello World ভ্যারিয়েশন বিভিন্নভাবে নাম প্রিন্ট করা।.py"
```
